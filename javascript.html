자바스크립트

자바스크립트는 사용자와 소통할 수 있다
<input> 태그에 속성으로 button을 주면 버튼 생성이 가능
value 속성의 값으로 주는 값이 버튼의 이름이 된다
onclick이라는 속성의 값으로는 자바스크립트가 와야한다
클릭시 자바스크립트의 내용이 실행
document.querySelector('body') <- body 태그를 선택하는 코드, 이후로 이어지는 내용을 body 태그에 넣어준다

이벤트


<input type = "button" value="hi" onclick="alert('hi')">
onclick이라는 속성 값으로는 반드시 javascript가 와야한다
onclick 속성의 속성 값은 웹브라우저가 기억하고 있다가 이 onclick 속성이 위치하고 있는 태그를 클릭했을 때 javascript의 코드가 실행
이렇게 웹브라우저 위에서 일어나는 일들을 사건, 또는 이벤트이라고 한다


<input type = "text" onchange="alert('changed')"> 텍스트 박스 안에 글씨가 추가하고 어딘가를 클릭gk면 onchange이라는 속성의 속성값인 javascript 코드가 실행


<input type = "text" onkeydown="alert('key down!')"> 텍스트 박스 안에서 키를 누르면 onkeydown이라는 속성의 속성값인 javascript 코드가 실행
이런 onclick, onchange, onkeydown 같은 것들을 event라고 한다

숫자

문자
문자는 따옴표로 시작해서 따옴표로 끝난다
작은 따옴표로 시작해서 작은 따옴표로 끝난다

"문자열길이세기".length
"소문자대문자로바꿔주기".toUpperCase()
"찾고자하는값의인덱스알려주기".indexOf('값')

변수와 대입 연산자

변수는 바뀔 수 있는 어떤 값
=은 대입 연산자, 오른쪽 항의 값을 왼쪽 변수에 대입한다
상수에는 수를 대입할 수가 없다
변수가 없다면 공식이라는 것을 사용할 수 없다
변수를 사용하면 바뀔 수 있는 공통적인 값이 아무리 많더라도 편리하게 바꿀 수 있다

함수

코드를 정리정돈하기 위한 코드

- 함수 만들기 -
function 함수명(매개변수){
 실행할 내용;
}

- 함수 실행하기 -
인자를 주어(없으면 안 줘도 된다) 함수를 호출
함수명(인자);

위에는 괄호가 있고, 아래는 괄호가 없다 똑같은 덧셈이라도 괄호 유무에 따라 값이 다르다
함수 호출시 괄호 안에 들어가는 것을 인자(argument)라고 하고, 함수 선언시 괄호 안에 들어가는 것을 매개변수(parameter)라고 한다
리턴 값에 괄호 안 써줬더니 오류 생겼다 undefined가 출력된다

이제 함수를 버튼에 적용해보자

누를 때 색은 바뀌어도 글씨는 안 바뀐다
함수 내의 this는 더 이상 input 버튼이 아니라 전역객체를 가리키게 되기 때문이다(웹브라우저에서는 윈도우)
그렇기에 함수 호출 시 인자로 this를 주고, 매개변수로 설정하여 이 값을 바꾸면 된다

객체

함수라는 것 기반 위에서 객체가 존재
객체 또한 정리정돈의 수단
서로 연관된 함수와 서로 연관된 변수를 같은 이름으로 그룹핑해주는 것
document는 객체, querySelector는 document에 속한 메소드
객체는 순서 없이 정보를 저장할 수 있는 것, 다만 마구잡이로 넣는 것은 아님
객체는 이름 있는 정리정돈 상자

- 객체를 생성 -
var 객체이름 = {
 "정보의이름":"정보"
}

객체에 접근할 때는 객체이름.정보의이름 혹은 객체이름["정보의이름"]
띄어쓰기가 있을 땐 .으로 접근할 수가 없다


객체는 중괄호로 표현, 아무것도 넣지 않으면 비어있는 객체
띄어쓰기가 있다면 괄호 안에 정보 이름을 넣으면 된다 넣을 때나 사용할 때나 마찬가지

- 객체에 들어있는 정보 모조리 가져오기 -
for(var key in itsMe){}
itsMe라는 변수가 가리키는 객체에 있는 key 값들을 가져오는 반복문
key값으로 정보의 이름이 나온다
그러므로 객체이름[key]하면 정보에도 접근할 수 있다


- 객체에 함수 추가하기 -
객체이름.함수이름 = function(){
 실행할 내용;
}


추가한 showAll까지 화면에 출력되었다
객체에 소속된 변수는 프로퍼티, 소속된 함수는 메소드라고 부른다
